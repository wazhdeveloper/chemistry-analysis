# 📊 化学成绩分析系统

> 一个高中化学教师自己用的成绩分析工具。每次考完试把 Excel 导进去，自动整合历史数据，生成成绩变化曲线和进退步分析。

---

## ✨ 功能

| 功能 | 说明 |
|:----|:------|
| 📥 **导入 Excel** | 支持新教育智能平台 + 智学网两种格式，自动识别，不用手动选 |
| 📈 **成绩趋势** | 按学生查看历次化学总分变化折线图 |
| 🎯 **题型分析** | 选择题/填空题得分率趋势对比（新教育平台数据） |
| 📉 **进退步分析** | 本次 vs 上次，分数变化 + 排名变化一目了然 |
| ⚠️ **退步预警** | 首页展示最近一次考试各班名次退步最大的前 5 名学生 |
| 🗑 **删除成绩** | 导错了直接删，重新导入 |
| 🔍 **搜索学生** | 按姓名搜索，快速查看个人成绩曲线 |

## 📂 Excel 格式

系统自动识别两种平台：

**新教育智能平台：**
```
学号 | 姓名 | 班级 | 总得分 | 客观题得分 | 主观题得分 | 班级排名 | 年级排名 | ...
```

**智学网：**
```
序号 | 准考证号 | ... | 班级 | 姓名 | 得分 | 校次 | 班次 | ...
```

> 如果 Excel 里没有选择题/填空题分，只填总分也能用，就是少了题型分析。

## 🛠 技术栈

| 东西 | 用啥 |
|:----|:------|
| 🐍 语言 | Python |
| 🖥 界面 | Streamlit |
| 🗄 数据库 | SQLite |
| 📊 图表 | Plotly |
| 📥 读 Excel | pandas + openpyxl + xlrd |

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install streamlit pandas plotly openpyxl xlrd

# 2. 启动
streamlit run app.py

# 3. 浏览器打开
# http://localhost:8501
```

### 部署到服务器

```bash
streamlit run app.py --server.port 3000 --server.address 0.0.0.0
```

然后通过 `http://服务器IP:3000` 访问。

## 📌 注意事项

- 化学满分固定 **100 分**
- 系统按 **学生姓名** 匹配跨次考试数据，同名会有提示
- 缺考自动识别（新教育平台不出现、智学网显示"未扫描"）
- 所有数据存在本地，不联网
