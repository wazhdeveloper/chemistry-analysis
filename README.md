# 📊 化学成绩分析系统

> 一个高中化学教师自己用的成绩分析工具。每次考完试把 Excel 导进去，自动整合历史数据，生成成绩变化曲线和进退步分析。

---

## ✨ 功能

| 功能 | 说明 |
|:----|:------|
| 📥 **导入 Excel** | 支持新教育智能平台 + 智学网两种格式，自动识别；考试类型可选（限时练习/月考/期中/期末/统考/自定义） |
| 📈 **成绩趋势** | 按学生查看历次化学总分变化折线图，标注缺考 |
| 🏆 **年级排名趋势** | 有年级排名数据时自动展示排名变化曲线（倒置 Y 轴） |
| 🎯 **题型分析** | 选择题/填空题得分率趋势对比（新教育平台数据） |
| 📉 **进退步分析** | 本次 vs 上次，分数变化 + 排名变化一目了然 |
| 🤖 **智能评语** | 基于历次成绩自动生成教师视角的综合分析评语（趋势、波动性、近期表现等） |
| ⚠️ **退步预警** | 首页展示最新考试各班退步学生，支持选择班级和调整显示人数 |
| 🔄 **连续退步预警** | 检测连续 2 次及以上成绩下降的学生，按班级分组展示 |
| 📊 **班级分析** | 完整的班级进退步总览：退步/进步明显分组、全班统计、分数段分布图、全班明细表、导出 CSV |
| 📈 **各班平均分趋势** | 首页展示各班级历次考试平均分变化曲线 |
| 🗑 **删除成绩** | 导错了直接删，重新导入 |
| 💾 **Excel 备份** | 导入时自动备份原始 Excel，支持编辑后重新读取更新 |
| 🔍 **搜索学生** | 按姓名搜索，快速查看个人成绩曲线 |
| 📤 **导出报告** | 学生成绩报告（TXT）+ 全班明细表（CSV） |

## 📂 Excel 格式

系统自动识别两种平台：

**新教育智能平台：**
```
学号 | 姓名 | 班级 | 总得分 | 客观题得分 | 主观题得分 | 班级排名 | 年级排名 | ...
```

**智学网：**
```
序号 | 准考证号 | ... | 班级 | 姓名 | 得分 | 校次 | 班次 | ...
```

> 如果 Excel 里没有选择题/填空题分，只填总分也能用，就是少了题型分析。

## 🛠 技术栈

| 东西 | 用啥 |
|:----|:------|
| 🐍 语言 | Python 3.8+ |
| 🖥 界面 | Streamlit |
| 🗄 数据库 | SQLite |
| 📊 图表 | Plotly |
| 📥 读 Excel | pandas + openpyxl + xlrd |

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动
streamlit run app.py

# 3. 浏览器打开
# http://localhost:8501
```

Windows 用户也可以直接双击 `run.bat` 启动。

## 📌 注意事项

- 化学满分固定 **100 分**
- 系统按 **学生姓名** 匹配跨次考试数据，同名会有提示
- 缺考自动识别（新教育平台不出现、智学网显示"未扫描"）
- 所有数据存在本地，不联网
- 编辑 Excel 后需点击「已编辑完成，重新读取更新」按钮来更新数据
